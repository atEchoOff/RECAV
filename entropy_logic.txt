For now, we will only consider periodic boundary conditions. Consider a discrete system of ODEs:
\begin{align*}\frac{d\fnt{u}}{dt}+\fnt{r}\aligned{=}\fnt{0}\labell{eq:DiscreteDG}\\\fnt{r}_{i}\aligned{=}\sum_{j}^{ }\left\lVert\fnt{n}_{ij}\right\rVert\vecf{f}\left(\fnt{u}_{i},\fnt{u}_{j},\frac{\fnt{n}_{ij}}{\left\lVert\fnt{n}_{ij}\right\rVert}\right)\labell{eq:DiscreteR}\end{align*}
with some given volume flux $\vecf{f}$. We aim to satisfy a discrete form of an entropy inequality:
\begin{align*}\frac{\partial\eta\left(\vecf{u}\right)}{\partial t}+\ub{\sum_{k}^{ }\frac{\partial F_{k}\left(\vecf{u}\right)}{\partial x_{k}}}{\nabla\cdot\vecf{F}}\le0\labell{eq:EntropyInequality}\end{align*}
Using \eqref{eq:DiscreteDG}, we can discretize $\frac{\partial\vecf{u}}{\partial t}$. To discretize the spatial derivative of the flux functions $F$, we first substitute in an entropy potential:
\begin{align*}F_{k}\left(\vecf{u}\right)=\vecf{v}^{T}\vecf{f}_{k}\left(\vecf{u}\right)-\psi_{k}\left(\vecf{u}\right)\end{align*}
where $\vecf{v}=\vecf{v}\left(\vecf{u}\right)$ are the entropy variables. We will now discretize the spatial derivative at point $\vecf{u}\left(\fnt{x}_{i}\right)=\fnt{u}_{i}$ using the same technique in \eqref{eq:DiscreteR}. However, rather than using the (potentially low order) flux $\vecf{f}$, we will restrict to the usage of the high order $\vecf{f}^{\text{central}}$, to obtain a "high order" discrete entropy inequality:
\begin{align*}\sum_{k}^{ }\frac{\partial F_{k}\left(\vecf{u}_{i}\right)}{\partial x_{k}}\gets\sum_{j}^{ }\left\lVert\fnt{n}_{ij}\right\rVert\left[\fnt{v}_{j}^{T}\vecf{f}^{\text{central}}\left(\fnt{u}_{i},\fnt{u}_{j},\frac{\fnt{n}_{ij}}{\left\lVert\fnt{n}_{ij}\right\rVert}\right)-\vecf{\psi}\left(\fnt{u}_{i},\fnt{u}_{j},\frac{\fnt{n}_{ij}}{\left\lVert\fnt{n}_{ij}\right\rVert}\right)\right]\end{align*}
At which point, the left hand side of \eqref{eq:EntropyInequality} at point $\vecf{u}\left(\fnt{x}_{i}\right)=\fnt{u}_{i}$ can be discretized using the following procedure (here we assume that $\vecf{f}=\vecf{f}^{\text{LxF}}$):
\begin{align*}\fnt{v}_{i}^{T}\frac{d\fnt{u}_{i}}{dt}+\sum_{k}^{ }\frac{\partial F_{k}\left(\vecf{u}_{i}\right)}{\partial x_{k}}\\-\fnt{v}_{i}^{T}\fnt{r}_{i}+\sum_{k}^{ }\frac{\partial F_{k}\left(\vecf{u}_{i}\right)}{\partial x_{k}}\\-\fnt{v}_{i}^{T}\sum_{j}^{ }\left\lVert\fnt{n}_{ij}\right\rVert\vecf{f}^{\text{LxF}}\left(\fnt{u}_{i},\fnt{u}_{j},\frac{\fnt{n}_{ij}}{\left\lVert\fnt{n}_{ij}\right\rVert}\right)+\sum_{j}^{ }\left\lVert\fnt{n}_{ij}\right\rVert\left[\fnt{v}_{j}^{T}\vecf{f}^{\text{central}}\left(\fnt{u}_{i},\fnt{u}_{j},\frac{\fnt{n}_{ij}}{\left\lVert\fnt{n}_{ij}\right\rVert}\right)-\vecf{\psi}\left(\fnt{u}_{i},\fnt{u}_{j},\frac{\fnt{n}_{ij}}{\left\lVert\fnt{n}_{ij}\right\rVert}\right)\right]\\-\fnt{v}_{i}^{T}\sum_{j}^{ }\left\lVert\fnt{n}_{ij}\right\rVert\left[\vecf{f}^{\text{central}}\left(\fnt{u}_{i},\fnt{u}_{j},\frac{\fnt{n}_{ij}}{\left\lVert\fnt{n}_{ij}\right\rVert}\right)-\frac{\lambda_{ij}}{2}\left(\fnt{u}_{j}-\fnt{u}_{i}\right)\right]+\sum_{j}^{ }\left\lVert\fnt{n}_{ij}\right\rVert\left[\fnt{v}_{j}^{T}\vecf{f}^{\text{central}}\left(\fnt{u}_{i},\fnt{u}_{j},\frac{\fnt{n}_{ij}}{\left\lVert\fnt{n}_{ij}\right\rVert}\right)-\vecf{\psi}\left(\fnt{u}_{i},\fnt{u}_{j},\frac{\fnt{n}_{ij}}{\left\lVert\fnt{n}_{ij}\right\rVert}\right)\right]\\\sum_{j}^{ }\left\lVert\fnt{n}_{ij}\right\rVert\left[\left(\fnt{v}_{j}-\fnt{v}_{i}\right)^{T}\vecf{f}^{\text{central}}\left(\fnt{u}_{i},\fnt{u}_{j},\frac{\fnt{n}_{ij}}{\left\lVert\fnt{n}_{ij}\right\rVert}\right)-\vecf{\psi}\left(\fnt{u}_{i},\fnt{u}_{j},\frac{\fnt{n}_{ij}}{\left\lVert\fnt{n}_{ij}\right\rVert}\right)+\frac{\lambda_{ij}}{2}\fnt{v}_{i}^{T}\left(\fnt{u}_{j}-\fnt{u}_{i}\right)\right]\end{align*}||{}
